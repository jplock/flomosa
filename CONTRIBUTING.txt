BUILD PROCESS
=============
- All users have read-only access to all repositories.
- Each repository is owned by a team and all members of the team have read-write access.
- User commits changes
- Changes are peer-reviewed, and the changes pushed to "pending" branch on ______.
- Hudson polls the "pending" branch on ______. Changes are merged to the "master" branch in the repository on Hudson. The build is performed.
- If the build is successful, the result is pushed to the "master" branch on GitHub.


CONTRIBUTING
============
- Fork repository on GitHub
- Make changes and commit to fork
- Send pull request to repository owners
- Repository owners pull changes into repository on GitHub.


NOTES
=====
- GitHub should host all repositories.
- Hudson repositories should only be used for testing the build.
